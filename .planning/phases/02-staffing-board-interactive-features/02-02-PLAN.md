<objective>
Implement the post-it notes feature for the staffing board. Users can create, edit, and delete colored post-it cards on project columns. Each post-it appears as a small card at the bottom of the project column with 3 color options (yellow, red, blue). Clicking opens a popover with textarea for editing and color picker buttons.

Purpose: Post-it notes give staffing meeting participants a way to track important context about projects directly on the board â€” visible at a glance without opening modals. The color system allows informal categorization (e.g., red for urgent) without formal category management.

Output: Post-it note cards on every project column with full CRUD, color selection, text truncation, and database persistence.
</objective>
